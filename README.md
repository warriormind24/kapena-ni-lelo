# Kapena Ni Lelo Poultry Farm

Company website for Kapena Ni Lelo Poultry Farm Ltd — Luanshya, Zambia.

## Run locally

Open `index.html` in a browser, or serve the folder:

```bash
npx serve .
```

## Deploy

Static files only. Upload the project folder to any web host, or enable **GitHub Pages** on the `main` branch (root directory).
